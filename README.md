# 📁 File Uploader

A full-stack file upload system using:

- Node.js
- Express.js
- MongoDB (GridFS)
- Multer
- HTML, CSS, JavaScript

## 🚀 Features
- Upload files
- Store in MongoDB (GridFS)
- View uploaded files
- Delete files
- Simple UI
